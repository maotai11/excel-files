# 📊 Excel Files Repository

這是一個專門用於存放和管理 Excel 檔案的倉庫。

## 📁 檔案結構

```
excel-files/
├── README.md
└── files/          # 存放所有 Excel 檔案
```

## 📥 如何使用

### 上傳檔案
1. 直接將 Excel 檔案放入 `files/` 目錄
2. Commit 並 Push 到 GitHub

### 下載檔案
直接從 GitHub 下載所需的 Excel 檔案
